# Track Suggester

Users will fill out a form. An Epicodus course track will be suggested based on users input.

## Getting Started

Clone project from Github repository, open index.html from top level of directory using any modern browser.

### Prerequisites

Any modern browser

* HTM/CSS/Bootstrap
